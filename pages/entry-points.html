Open Source Contributions

This is underrated for building credibility:

  • Fashion-MNIST (Zalando) -- contribute improvements, benchmarks
  • DeepFashion tools and extensions
  • Open-source recommendation engines used by fashion companies
  • Build and open-source your own tools (outfit recommender, trend scraper)

Content Creation as Networking

The fashion tech niche is small. Consistent posting gets noticed fast.

  • Post weekly on LinkedIn about fashion tech trends
  • Start a short blog/newsletter
  • Make TikTok/YouTube explainers on fashion tech -- almost nobody does this
  • This builds your reputation before you apply anywhere

Fashion-Specific Hackathons

  • Fashion Hackers (NYC, annual)
  • Hacking Fashion (London)
  • Fashion Tech Forum Hackathon
  • CLO Virtual Fashion design competitions
  • General hackathons with fashion tracks (LA Hacks, HackMIT)
